The Security Awareness Library Tool (SALT) is an educational web application that contains a library full of definitions, articles and videos related to cyber security. It was created to provide guidance for Information Technology students who are studying cyber security terminology and concepts. This application has the following features: a search feature, a 
“request to add term” feature, a flashcard creator, a data scraping tool, and a usage metrics counter. This was all created using the following: HTML, CSS, JavaScript, PHP, MongoDB queries, Microsoft Azure’s App Service, Twilio’s SendGrid mail server, Python, the Django framework and the Selenium framework.
